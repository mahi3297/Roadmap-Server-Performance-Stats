# server-stats.sh - Linux Server Performance Monitor

A portable Bash script that provides comprehensive server performance statistics. Works on any Linux distribution without additional dependencies.

## Features

✅ **Core Metrics**
- Total CPU usage percentage
- Memory usage (Free vs Used with %)
- Disk usage (Free vs Used with %)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage

✅ **Bonus Metrics (Stretch Goals)**
- OS version/distribution
- System uptime & load averages
- Currently logged-in users
- Failed login attempts (last 24h)

## Quick Start

```bash
# 1. Save the script below as server-stats.sh
# 2. Make executable
chmod +x server-stats.sh

# 3. Run
./server-stats.sh
